pub fn create_board_struct() -> Board
Expand description

It returns a ‘board’ structure that we set earlier. Details are below.

It is the same as the code below:
let board = Board {
    grid: [[None, None, None], [None, None, None], [None, None, None]],
    current_turn: Player::X,
    winner: None
};
run_game(board);
We use it this way instead of the above code:
run_game(create_board_struct());
